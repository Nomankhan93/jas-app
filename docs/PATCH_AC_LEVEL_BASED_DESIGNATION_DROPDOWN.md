# Patch AC — Level-based designation dropdown

## Change

The Assign Designation form now shows professional designation options based on the selected level.

  - Vice Chairman
  - General Secretary

- Taluka

  - Taluka Finance Secretary

## Notes

- Existing configured designations are still included in the dropdown and marked as `configured`.
- The final card title remains editable, so admins can fine-tune the title before saving.
- No database migration.
